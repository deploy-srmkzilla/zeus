# Installation 🔧

## Install dependencies

```
$ yarn
```

## Build the server

```
$ yarn build
```

## Start the server

```
$ yarn preview
```

## Start the development server

```
$ yarn dev
```

<br>

### Note: You must add your own enviroment variables. Checkout the `env.example` and make a `.env` file before starting the application.
